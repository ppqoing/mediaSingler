# GUI 配置、审阅与删除模块计划

## 1. 模块目标

把当前截图演示界面演进为多磁盘去重控制台。GUI 集中管理扫描路径、线程、I/O、队列、重试、超时、RocksDB、MySQL、缩略图尺寸与缓存和日志配置；展示本地与远端进度；分页审阅精确/相似组；通过批量条件选择后执行永久删除。所有耗时 I/O、FFmpeg、数据库和文件删除都在后台执行。

## 2. 页面结构

### 2.1 新建扫描

- 路径列表：浏览添加、手工输入、删除、拖动排序。
- 显示每条路径对应的物理盘、在线状态和路径优先级。
- 扫描所有普通文件并计算 SHA-512 的固定说明。
- 开始前显示 RocksDB、MySQL、线程、I/O、超时、同步批量、缩略图尺寸和资源预算摘要。

### 2.2 运行监控

- 任务状态、阶段、暂停、继续、取消。
- 每物理盘卡片：介质、读取线程、队列、当前路径、吞吐和错误。
- HDD 卡片显示当前电梯方向、排序窗口、物理区间命中率、坏块/超时文件数和 `CancelPending` 工作者。
- 全局计算线程：配置值、活动数和排队任务。
- RocksDB：写入状态和本地检查点。
- MySQL：在线/离线、待同步量、最后成功时间和错误。
- 明确区分“本地扫描完成”和“共享数据同步完成”。

### 2.3 精确重复

- 自动显示 MySQL SHA-512 相同组。
- 分页、排序和筛选物理盘、路径、成员数和空间。
- 展示路径、SHA-512、大小、时间、硬链接和保留成员。

### 2.4 内容相似

- 独立“计算内容相似”按钮，由用户手动触发。
- 图片组展示图片缩略图和 dHash 距离。
- 视频组展示每个成员的整张 2×3 拼图、时长、六组平均距离、分辨率和码率。
- 静态画面视频单独标识，不进入视频相似结果。
- 使用分页和按需缩略图，不全量加载。

### 2.5 历史任务和错误

- 扫描、同步、匹配和删除批次状态。
- 恢复 `Interrupted` 任务。
- 查询单文件错误、数据库错误和待重试项。
- 打开滚动日志目录。

### 2.6 诊断工具

当前单视频截图、单文件 SHA-512、dHash、汉明距离和磁盘查询移到诊断页；耗时操作也通过后台任务执行。

## 3. 配置页面

- 配置固定保存到程序安装目录下的 `config.json`，页面显示实际绝对路径和当前 `schema_version`。
- MySQL 密码使用 DPAPI `CurrentUser` 加密，密文 Base64 写入 JSON；密码框默认隐藏，界面、日志和错误信息不显示明文。
- DPAPI 解密失败时保留其他设置，密码框显示“需要重新输入”，测试连接和初始化按钮保持禁用，直到用户输入并成功保存新密码。
- “保存配置”先执行范围、目录、线程总数和资源预算校验，再写 `config.json.tmp` 并原子替换主文件，同时保留 `config.json.bak`；校验或写入失败时不破坏旧文件。
- 安装目录不可写、磁盘空间不足或替换失败时显示具体路径和系统错误，不请求静默提权，也不改存其他目录。
- 主文件缺失时显示内置默认值；主文件损坏时提示并尝试读取备份，未经用户确认不覆盖损坏文件。
- 创建任务时把页面值转换为不可变 `ScanOptions` 并写入 RocksDB；运行中修改只影响下一任务，中断恢复使用原快照。
- dHash 阈值、视频采样点、2 秒时长规则和静态画面规则以只读算法版本展示，不提供普通编辑控件。

### 3.1 RocksDB

- 持久化数据库目录。
- 缓存和写缓冲档位。
- 查看大小、Column Family 和最后检查点。
- 有活动任务时禁止切换目录。

### 3.2 MySQL 8.0+

- 主机、端口、数据库名、用户名、密码。
- TLS 模式、CA、证书和密钥路径。
- 连接池大小、连接和命令超时、重试间隔、同步批量大小。
- `mysqldump` 路径和备份目录。
- “测试连接”按钮。
- “初始化数据库表”按钮。

不提供“一键重建”“清空表”或删除数据库按钮。密码控件默认隐藏，保存时使用 Windows 本机保护。

### 3.3 缩略图与缓存

- 缩略图根目录。
- 文件格式，默认 JPEG。
- 视频 2×3 拼图单格长边，默认 256 像素。
- 图片结果预览缩略图长边，独立于 dHash 的固定 9×8 输入尺寸。
- 缩略图缓存项数、内存上限和显存上限。
- 路径可写检查和测试生成。
- 修改格式或尺寸时显示“已有缩略图将按需重建”；新规格版本不触发 SHA-512、dHash 或媒体元数据重算。

### 3.4 线程

- 按物理盘列出 HDD/SSD/Unknown、关联卷和读取线程数。
- HDD 默认 1，SSD 默认 2，用户可修改。
- 全局计算线程数。
- FFmpeg 单任务内部线程数。
- 显示预计总读取线程和内存缓冲预算。
- 设置只对下一任务生效。

### 3.5 I/O、队列与超时

- 读取块大小和每物理盘文件队列容量。
- HDD 显示“物理区间优化”开关和排序窗口大小；SSD 默认禁用该选项。
- 可配置坏块原块重试次数、小块重试次数、小块尺寸和无进展超时；默认超时 60 秒。
- 明确提示坏块只跳过文件，不自动暂停整盘。
- 所有数字控件显示单位、合法范围和推荐值；预计内存超过安全预算时警告，非法值禁止启动任务。

### 3.6 日志

- 日志目录、单文件大小、滚动数量和保留天数。
- 删除前测试日志可写。
- 日志不显示 MySQL 密码或 TLS 私钥。
- 坏块日志可按物理盘、路径、偏移、错误码和超时筛选。

### 3.7 千万级索引

- 显示精确索引、图片 dHash 索引和视频 dHash 索引的版本、内容数、磁盘占用和构建进度。
- 手动内容相似任务开始前估算 RocksDB 可用空间。
- 支持暂停/继续索引，不提供绕过索引执行全量两两比较的选项。

## 4. 路径列表交互

- “添加路径”打开目录选择器。
- “删除路径”只从下一次扫描配置移除，不删除数据库历史。
- 拖动排序决定路径保留优先级。
- 重复、包含关系、离线盘和无权限路径在开始前提示。
- 任务运行时路径列表只读，避免当前快照变化。
- 历史路径状态显示 `Missing`、`Offline`、`OutOfScope`。

## 5. 批量选择界面

下列控件只改变勾选状态：

- 选择较大文件。
- 选择低质量图片/视频。
- 选择指定物理磁盘。
- 保留最新、最旧、最小、最大。
- 按路径优先级保留。
- 全部取消选择。

每个组最后一个保留成员不可选。系统自动取消时显示原因。SHA-512 精确组的“选择较大文件”禁用，因为成员大小相同。

## 6. 永久删除界面

- “永久删除选中文件”与所有选择条件分离。
- 点击后打开全屏或模态预览，显示不可恢复警告。
- 显示组数、文件数、释放空间、物理盘、路径和强制保留成员。
- dHash 组必须显示可见成员的图片或完整 2×3 拼图。
- 用户二次确认后发送不可变删除命令。
- 运行中显示本地删除和 MySQL 映射删除两个进度。
- 部分失败时列出文件错误和待同步 tombstone。

## 7. 缩略图内存和显存管理

### 7.1 加载策略

- 数据库只返回缩略图路径，不返回 BLOB。
- 只为当前可见行及少量预取行异步读取文件。
- 视频只加载一张 2×3 拼图，不加载六张独立图。
- 后台线程解码普通像素；GUI 线程创建 D3D11 纹理。

### 7.2 释放策略

- 使用有界 LRU，按纹理字节数和条目数双重限制。
- 行离开可见区后降低优先级；超过预算立即释放最旧纹理。
- 切换任务、关闭结果页或数据库重连不保留无关纹理。
- D3D11 纹理只在 GUI 线程释放。
- 提供当前缓存条目、内存和显存统计用于诊断。

## 8. UI 与后台边界

- GUI 发送不可变命令，协调器验证任务 ID 和状态。
- 进度通过每 100 至 250 ms 的不可变快照更新。
- RocksDB/MySQL 查询由分页服务执行。
- UI 不直接遍历 RocksDB、不持有 MySQL 事务、不执行永久删除。
- 命令执行结果包含受理、拒绝和中文原因。

## 9. 大数据量展示

- ImGui 表格使用 clipper 和数据库分页。
- 排序、筛选在查询层完成。
- 路径只绘制可见文本，悬停或复制提供完整值。
- 2×3 拼图按可见行加载。
- 进度图使用固定长度环形采样。
- 错误先按类型聚合，再按需查询明细。

## 11. 实施任务

### M1：配置

- 路径排序、RocksDB、MySQL、缩略图尺寸与缓存、线程、I/O、队列、重试、超时和日志页面。
- 连接测试与初始化表按钮。
- 配置校验、资源预算预估、任务快照和缩略图规格版本提示。
- 安装目录 `config.json`、模式迁移、DPAPI、临时文件原子替换、有效备份和损坏恢复。

### M2：任务监控

- 命令、状态、逐盘、HDD 物理顺序、坏块/超时、计算、RocksDB 和 MySQL 同步快照。

### M3：精确结果

- SHA-512 组分页、筛选、成员和空间。

### M4：内容相似

- 手动触发、千万级索引进度、图片缩略图、视频整张 2×3 拼图和距离证据。

### M5：LRU 纹理缓存

- 后台像素加载、UI 线程纹理、字节/条目预算和及时释放。

### M6：选择与永久删除

- 条件选择、强制保留、预览、二次确认和双阶段进度。

### M7：历史和恢复

- 中断恢复、错误、待同步项和日志入口。

## 12. 验收标准

1. GUI 可添加、删除和排序扫描路径。
2. 可为每个物理盘配置读取线程，并配置全局计算线程和 FFmpeg 单任务线程。
3. 可配置读取块、每盘队列、HDD 排序窗口、坏块重试、小块尺寸和无进展超时。
4. MySQL 配置支持 TLS、连接/命令超时、重试间隔、同步批量、测试连接、初始化表、mysqldump 和备份目录。
5. 可配置视频拼图单格尺寸、图片预览尺寸和缩略图内存/显存预算，规格变化能够按需重建。
6. 非法配置不能启动任务；运行中修改不改变当前任务，恢复任务使用原配置快照。
7. 配置重启后从安装目录 `config.json` 恢复，保存失败不会破坏上一有效配置。
8. MySQL 密码在 JSON、RocksDB、日志和界面中均不以明文持久化或输出。
9. 没有数据库重建或清空按钮。
10. 本地完成和 MySQL 同步完成清晰区分。
11. 视频结果展示整张 2×3 拼图。
12. 大量结果只加载可见缩略图，LRU 超限后及时释放纹理。
13. 选择条件不会直接删除文件。
14. 最后一个保留成员不能被选中。
15. 永久删除有完整预览和二次确认。
16. UI 线程不执行文件扫描、哈希、FFmpeg、MySQL 或删除操作。
17. 关闭活动任务能够安全保存 RocksDB 检查点。
18. 坏块和超时文件可在 GUI 定位日志，但不会让整盘自动进入暂停状态。
19. GUI 可显示千万级索引进度、占用和断点状态。

## 13. 风险与对策

| 风险 | 对策 |
| --- | --- |
| dHash 组误删 | 拼图/缩略图、距离、分离选择与删除、二次确认 |
| 拼图耗尽显存 | 可见行加载、有界 LRU、UI 线程释放 |
| 配置过多难理解 | 分区、默认值、测试按钮和任务摘要 |
| MySQL 离线被误认为完成 | 双进度和明确离线待同步状态 |
| 永久删除误触 | 独立危险按钮、不可恢复提示、二次确认和保留硬约束 |
| HDD 坏道造成界面像卡死 | 显示无进展计时、取消状态、错误偏移和后续文件进度 |
| 内容索引占满 RocksDB 盘 | 开始前空间估算、运行中占用显示和有界分片 |
| 安装目录无写权限 | 保存前可写检查、显示绝对路径和 Win32 错误；部署时选择可写安装目录或配置明确 ACL |
| JSON 损坏或保存中断 | 临时文件、刷盘、原子替换、上一有效备份和显式恢复提示 |
| DPAPI 密文无法解密 | 其他配置继续加载，数据库操作禁用并要求重新输入，不自动覆盖旧密文 |
