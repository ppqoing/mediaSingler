# 图片三级相似稳定验收语料

这些 PPM 文件由固定数学图案离线生成后直接纳入仓库；测试运行时只读取文件，不联网，也不调用图片生成器创建验收输入。

- `base.ppm`：基准主内容。
- `resolution-128x96.ppm`：相同构图的 2× 分辨率版本。
- `brightness.ppm`：亮度变化。
- `warm-tone.ppm`：暖色调变化。
- `watermark.ppm`：局部半透明文字条形水印。
- `hard-negative.ppm`：不同主内容的难负样本。

验收约束：四个正变体必须依次通过 PDQ、4×4 pHash 和结构三筛；难负样本必须至少被其中一级拒绝。水印样本用于校准一筛候选召回与结构全局分下限，最终仍必须通过分区 pHash、结构裁剪分和通过块比例。旋转、裁剪、平移、局部重复和语义相似不属于正样本。
